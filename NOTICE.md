# AeroLang — Public Portfolio Repository

This is a **public portfolio / demo version** of AeroLang.

Certain implementation files (compiler internals and runtime library `.cpp` sources)
are intentionally omitted or replaced with stubs to protect proprietary logic.

The visible codebase demonstrates:
- Language design (`.aero` example programs in `examples/`)
- Compiler architecture (`include/*.h` headers, `CMakeLists.txt`)
- VS Code extension (IntelliSense, Build system, Device Manager, Debugger)
- Build tooling (`tools/aero-build`, `aero-init`, `aero-gradle`)
- Runtime API surface (`runtime/*.h` headers for all 30+ APIs)
- Full documentation (`docs/`, `README.md`, progress reports)
